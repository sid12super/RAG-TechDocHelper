# Contribution Guidelines

While this is primarily a personal portfolio project, I welcome feedback and suggestions. If you find a bug or have an idea for an improvement, please feel free to open an issue first to discuss it.

## Development Setup

1.  Fork the repository.
2.  Clone your fork: `git clone https://github.com/your-username/RAG-TechDocHelper.git`
3.  Install the required dependencies: `pip install -r requirements.txt`

## Submitting Changes

-   Please ensure your code passes the CI checks before submitting a pull request.
-   The CI pipeline automatically runs `flake8` for linting and `black` for formatting.
